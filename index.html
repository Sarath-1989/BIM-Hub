📋Setup Guide

One-time setup by your IT admin. Once done, all 30+ team members can log in with their Microsoft 365 accounts.

1
Create SharePoint ListsGo to your SharePoint site → Site Contents → New → List. Create BIMProjects and BIMTasks.
2
BIMProjects columnsAdd: ProjectCode, Client, Status, Phase, StartDate, EndDate, Progress, Color, TeamMembers (person, multiple)
3
BIMTasks columnsAdd: ProjectID, Phase, Discipline, Assignee (person), StartDate, EndDate, Priority, Status, Progress, Notes
4
Register Azure AppGo to portal.azure.com → Azure AD → App Registrations → New. Grant Sites.ReadWrite.All. Copy Tenant ID and Client ID above.
4b
BIMMembers List columnsAdd these columns to your BIMMembers SharePoint list: Email (Single line), SystemRole (Single line), BIMData (Multiple lines). Admin can assign roles from Settings → User Role Management.
5
Deploy for your teamUpload this file to your SharePoint Document Library or company intranet. Everyone with a Microsoft 365 account can access it — no additional software needed.
