# Changelog

All notable changes to the Asterwix Unified Web Portal will be documented in this file.

## [1.0.0] - 2026-06-03
### Added
- Initial Release of the Unified Web Portal architecture.
- Single Page Application UI structure in `portal.html`.
- MSAL.js v3 Active Directory integration for Single Sign-On (SSO).
- Interactive Mock Mode bypass with role-switcher for evaluation.
- GPS Punching Attendance module with Leaflet.js map, geofencing, distance indicators, and manual overrides.
- Project Hub (BIM) module with progress dials, project logs, drawing reviewer logs, timeline Gantt chart, and task board.
- HR & Accounts module with employee directory, leave planner, overtime calculators, WPS SIF export, self-service payslip viewer (print ready), and profile page.
- Session timeout safeguards and local storage persistence.
- SWA configuration files and deployment actions workflow.
